# **Практична робота №4: Розробка REST API для енергетичних даних (Node.js \+ Express)**

## **Інструкція із запуску**

1. Переконайтеся, що на вашому комп'ютері встановлено **Node.js**.  
2. Відкрийте термінал (або командний рядок) безпосередньо в кореневій папці проєкту.

Встановіть необхідні модулі (фреймворк Express), виконавши команду:

Bash  
npm install  
npm install express

Запустіть локальний сервер командою:

Bash  
node server.js

3. Дочекайтеся повідомлення server run: http://localhost:3000 у терміналі.  
4. Для перевірки роботи API відкрийте другу вкладку терміналу та запустіть скрипт автоматичного тестування командою node test.js,.
